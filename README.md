# SillyTavern Character Dialogue Colors

Automatically colors character names in dialogue without requiring HTML tags in context.

## Features

- Automatically detects character names in messages
- Assigns persistent colors to each character
- Works without modifying chat context
- Compatible with dark and light themes

## Installation

1. Download or clone this repository
2. Place the folder in your SillyTavern `public/scripts/extensions/` directory
3. Restart SillyTavern or reload extensions

## Usage

The extension automatically detects character names at the start of messages (format: "Name:" or similar) and applies consistent colors. No configuration needed.

## Default Colors

- Hot Orange (#FF6B35)
- Bright Pink (#FF1493) 
- Dark Turquoise (#00CED1)
- Lime Green (#32CD32)
- Gold (#FFD700)
- Hot Pink (#FF69B4)
- Blue Violet (#8A2BE2)
- Orange Red (#FF4500)

Colors cycle through this list for new characters.
